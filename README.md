# Бот Посланий Дня

Telegram-бот для ежедневных посланий с энергией для вдохновения и поддержки.

## Описание

Бот предоставляет пользователям:
- **Послание дня** - уникальное послание, одинаковое для каждого пользователя в течение дня
- **Случайное послание** - случайное послание в любое время
- Каждое послание сопровождается визуальным образом для медитации

## Требования

- Go 1.19 или выше
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
- Изображения посланий в формате JPG (например: `1.jpg`, `2.jpg`, и т.д.)

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd messages-for-the-day-bot
```

2. Установите зависимости:
```bash
go mod download
```

3. Настройте конфигурацию:
   - Откройте файл `config.yaml`
   - Замените `YOUR_BOT_TOKEN_HERE` на ваш токен бота от BotFather
   - Или установите переменную окружения `BOT_TOKEN`

## Настройка

### Способ 1: Через config.yaml

Отредактируйте файл `config.yaml`:

```yaml
bot:
  token: "YOUR_BOT_TOKEN_HERE"  # Замените на ваш токен
  debug: true                    # Режим отладки
  timeout: 60                    # Таймаут обновлений в секундах

messages:
  welcome: |
    Ваше приветственное сообщение...
  
images:
  path: "images"                 # Путь к папке с изображениями
  daily_reminder: true           # Включить ежедневные напоминания
```

### Способ 2: Через переменные окружения (рекомендуется)

Установите переменную окружения `BOT_TOKEN`:

**Windows (PowerShell):**
```powershell
$env:BOT_TOKEN="your_bot_token_here"
```

**Windows (CMD):**
```cmd
set BOT_TOKEN=your_bot_token_here
```

**Linux/macOS:**
```bash
export BOT_TOKEN="your_bot_token_here"
```

Приоритет: переменная окружения имеет приоритет над значением в `config.yaml`.

## Подготовка изображений

1. Создайте папку `images` в корне проекта
2. Добавьте изображения в формате JPG с числовыми именами:
   - `1.jpg`
   - `2.jpg`
   - `3.jpg`
   - и т.д.

Бот автоматически загрузит все доступные изображения при запуске.

## Запуск

```bash
go run .
```

Или скомпилируйте и запустите:

```bash
go build -o bot
./bot
```

## Команды бота

- `/start` - Начать работу с ботом
- `/card` - Получить послание дня
- `/random` - Случайное послание
- `/help` - Помощь и справка
- `/about` - Информация о боте

## Безопасность

⚠️ **ВАЖНО**: Никогда не коммитьте файл `config.yaml` с реальным токеном в публичный репозиторий!

Рекомендуется:
- Использовать переменные окружения для токена
- Добавить `config.yaml` в `.gitignore` (если содержит чувствительные данные)
- Использовать `.env` файлы для локальной разработки (не коммитить их)
